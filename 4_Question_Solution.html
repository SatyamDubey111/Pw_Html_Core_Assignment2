List of HTML tags with information.

  1. h1 to h6 : Define heading tags with decreasing levels.
  2. p: Define a paragraph of a text.
  3. a: Create a hyperlinks.
  4. table: Define an HTML table with row and column.
  5. ul: It's Define a unordered list.
  6. body: It's Define as a container of a html documents.
  7. button: Defines as a click able button with in a form.
  8. br: This tag is used for break line of text.
  9. hr: This tag is used for draw a horizontal line.
  10. img: This tag is for brining image on the web
