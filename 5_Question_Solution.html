Full Stack Web Development Tech Stacks.

Html, Css, JavaScript:
These three are fundamental for building the structure, style , and interactivity of web pages.
React:
This is a JavaScript library for building user interfaces .
Node.js:
Node.js is an open source cross platform, javascript runtime that allows you to execute server side code.
Express.js:
This is a web application framework for node.js, simplifying the creation of server side logic and APIs.
Mongo DB:
Mongo DB is a No SQL document oriented database program, often used for it's flexibility and scalability.
My SQL:
This is a relational database management system, commonly used for structured data.
